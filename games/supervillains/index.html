Supervillains

This is a temporary webpage to host a preliminary playable version of supervillains.

How to Play

Join:

Start [#]: Start a game with a specific number of players.
Join [###]: Joins a game using it's Game ID.

Commands:

Draw <#>: Draw a card. Use a number to draw a specific amount.
Play [ID] [Stack]: Plays a card into a specific stack.
Spring [ID]: Springs a card.
Info [#]: Get the info for an opponent.
Steal [#]: Steals a random card from an opponent.
Clear [#]: Clears a specific stack.
